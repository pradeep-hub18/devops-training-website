Detailed syllabus

Step-by-step modules for real project confidence.

Linux and shell basics

Commands, file permissions, users, services, packages, logs, networking tools, processes, and shell scripting fundamentals.

Git and GitHub workflow

Repositories, branches, commits, pull requests, merge conflicts, tags, releases, and team collaboration workflows.

Jenkins CI/CD

Freestyle jobs, pipelines, credentials, agents, webhooks, build stages, artifacts, test reports, and deployment automation.

Docker containers

Images, Dockerfiles, registries, volumes, networks, Compose, best practices, scanning, and debugging failed containers.

Kubernetes core

Pods, ReplicaSets, Deployments, Services, ConfigMaps, Secrets, namespaces, probes, resource limits, and rollout strategies.

Helm and release management

Charts, values, templates, releases, upgrades, rollbacks, environment-specific configuration, and reusable deployment patterns.

ArgoCD GitOps

GitOps principles, applications, sync policies, app health, drift detection, rollback, and multi-environment delivery.

Terraform IaC

Providers, variables, outputs, state, modules, workspaces, remote backends, planning, applying, and safe change review.

AWS and EKS

IAM, VPC, EC2, ECR, Load Balancers, Route 53 basics, EKS clusters, node groups, kubectl access, and deployment on AWS.

Monitoring and logs

Metrics, alerts, dashboards, logs, application health, Prometheus and Grafana concepts, and incident visibility.

Troubleshooting labs

ImagePullBackOff, CrashLoopBackOff, pending pods, failed probes, ingress issues, broken pipelines, IAM errors, and networking faults.

Interview preparation

Resume positioning, scenario questions, project explanation, mock interview practice, and common DevOps interview answers.

FAQ

Common questions before joining.

Basic scripting helps, but the course starts with Linux and command-line foundations before moving into pipelines and cloud deployment.

Will I work on real projects?

Yes. The course is designed around hands-on labs, CI/CD workflows, container deployments, EKS, GitOps, and troubleshooting scenarios.

Is AWS included?

Yes. AWS fundamentals, ECR, EKS, IAM basics, load balancers, and Terraform-based cloud provisioning are part of the learning path.

Will interview preparation be covered?

Yes. You will practice scenario questions, project explanations, resume positioning, and common DevOps interview topics.
